# 520表白网页

这是一个充满爱意的表白网页，包含以下特色功能：

- 💝 实时计时展示恋爱天数
- 🌸 浪漫的飘动爱心动画
- 📱 支持上传小程序码
- 🖼️ 照片墙功能
- 💌 甜蜜的告白文字
- 📅 未来约定展示
- 📱 完美支持手机端显示

## 在线预览

访问 GitHub Pages 地址即可查看网页效果。

## 本地运行

1. 克隆本仓库
2. 使用任意 HTTP 服务器运行，例如：
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - PHP: `php -S localhost:8000`

## 自定义修改

1. 修改 `index.html` 中的文字内容
2. 调整 `style.css` 中的样式
3. 更新 `script.js` 中的计时起始日期
